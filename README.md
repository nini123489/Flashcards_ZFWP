# ZFWP Flashcards

Interaktive Karteikarten-Web-App für das Fach "Zerstörungsfreie Werkstoffprüfung" (bei prof. Kasperl) mit Fragen und Antworten. Die Seite ist komplett statisch und kann lokal im Browser betrieben werden. Eine bereits gehostete Version findest du hier: [https://zfwp.nicklaseckardt.de](https://zfwp.nicklaseckardt.de).
Alle Angaben und Inhalte sind ohne Gewähr und können Fehler enthalten.

## Lokal ansehen

```bash
# Repository klonen
git clone https://github.com/nini123489/Flaschcards_ZFWP.git
cd Flaschcards_ZFWP

# index.html im Browser öffnen
```
